# Luo Manenos (LM-OS)

Luo Manenos is an entirely new, open-source Linux-based operating system built for **Privacy** and **Gaming**.

## Features

- **Gaming Optimized Kernel:** Integrated with Liquorix/Zen patches for low-latency responsiveness.
- **Privacy Hardened:** Kernel-level security hardening and privacy-focused default configurations.
- **De-Googled:** Built-in support for MicroG as a privacy-respecting alternative to Google Play Services.
- **Open Source:** Built entirely from publicly available source code under the GPL and other open-source licenses.

## Getting Started

Luo Manenos is currently in the prototype stage. You can build your own image using the provided Buildroot configurations.

### Build Instructions

1. Clone this repository: `git clone https://github.com/your-username/luo-manenos.git`
2. Download Buildroot: `make download-buildroot`
3. Apply the Luo Manenos configuration: `make apply-config`
4. Build the image: `make build`
